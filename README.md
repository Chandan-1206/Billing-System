# Billing System

**Simple, modular Python billing system** — generate invoices, manage inventory, and export reports.

## Features
- Create / update / delete customers and products  
- Create invoices with line items, taxes and discounts  
- Save invoices (JSON / CSV) and export printable PDF (if enabled)  
- Basic reporting: sales by customer, date-range summary  
- Simple CLI and REST endpoints (if `app.py` / `server.py` present)
